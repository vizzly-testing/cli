Sample Static Site Page

Demonstrating Vizzly screenshot capture with the companion config files

About This Page

This page is discovered like any other HTML file. Viewports and interactions are configured in the companion vizzly.config.js and vizzly.static-site.js files.

Viewport Configuration

The example config captures only mobile and desktop viewports for this page.

Interaction Hooks

The example interactions file references a named hook that scrolls to the footer before the screenshot is taken.

Features

  • Automatic page discovery from sitemap.xml
  • Direct HTML file scanning
  • Per-page viewport configuration
  • Interaction hooks for dynamic content
  • Pattern-based filtering
  • TDD and Run mode support
