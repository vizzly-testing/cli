Demonstrating Vizzly screenshot capture with meta tag configuration

About This Page

This page demonstrates how to configure Vizzly screenshot capture using HTML meta tags.

Viewport Configuration

This page uses custom viewport configuration:

<meta name="vizzly:viewports" content="mobile,desktop">

Only mobile and desktop viewports will be captured.

Interaction Hooks

This page references a named interaction hook:

<meta name="vizzly:interaction" content="scroll-to-footer">

The page will scroll to the footer before the screenshot is taken.

Features

  • Automatic page discovery from sitemap.xml
  • Direct HTML file scanning
  • Per-page viewport configuration
  • Interaction hooks for dynamic content
  • Pattern-based filtering
  • TDD and Run mode support
